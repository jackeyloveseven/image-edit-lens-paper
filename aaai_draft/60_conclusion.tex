\section{Conclusion}\label{sec:conclusion}

A DiT image editor under fixed underspecified prompts carries the realized
edit as an early target-image code and translates it to the velocity
convention only in L52--54; probes, tuned/differential lenses, and
interventions (transplant, steering) bound what that code predicts and can
write. Held-out selection and related checkpoints support the readout; CFG
truncation after step two is a training-free lever on the same map. We
claim only the tested families and gates above, not irreversible
commitment or architecture-wide necessity.
